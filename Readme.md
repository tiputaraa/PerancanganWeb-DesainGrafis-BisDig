# PerancanganWeb-DesainGrafis-BisDig

## Studi Kasus: Tampilkan Komentar Dinamis dengan JavaScript

Pada studi kasus ini, kita membuat tombol "Tampilkan Komentar" agar komentar tidak langsung terlihat, melainkan baru muncul saat tombol diklik.

### Cara kerja:

- Komentar disembunyikan terlebih dahulu dengan CSS (`display: none`).
- JavaScript menangani event klik tombol untuk menampilkan atau menyembunyikan komentar secara dinamis.
- Tulisan pada tombol berubah sesuai dengan keadaan komentar (Tampilkan atau Sembunyikan).

### Manfaat:

- Meningkatkan tampilan halaman agar lebih rapi.
- Pengguna dapat memilih apakah ingin melihat komentar atau tidak.
- Contoh sederhana penggunaan JavaScript untuk interaktivitas pada web statis.

### Cara menggunakan:

1. Buka file `index.html` di browser.
2. Klik tombol "Tampilkan Komentar" untuk melihat komentar.
3. Klik lagi untuk menyembunyikan komentar.
