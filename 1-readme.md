# Halaman Profil Fotografer

## Studi Kasus
Diminta membangun halaman profil untuk seorang fotografer profesional yang memuat gambar, deskripsi, dan tautan media sosial.

## Struktur HTML5
- `<header>`: Menampilkan judul halaman.
- `<main>`: Konten utama yang berisi tiga bagian:
  - `<section>` berisi gambar utama fotografer.
  - `<section>` untuk deskripsi "Tentang Saya".
  - `<section>` untuk tautan ke media sosial fotografer.
- `<footer>`: Berisi informasi hak cipta.

File `profil.html` ini merupakan implementasi struktur HTML5 yang rapi dan responsif.
