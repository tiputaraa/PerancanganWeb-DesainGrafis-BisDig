# PerancanganWeb-DesainGrafis-BisDig

## Studi Kasus: Landing Page Promosi + Prototipe Figma

### Tujuan:
Perusahaan ingin membuat landing page promosi menggunakan framework. Sebelum implementasi, desainer diminta membuat prototipe menggunakan Figma.

### Manfaat Membuat Prototipe (Figma) Sebelum Coding:
- Bisa dilihat klien untuk validasi visual.
- Lebih mudah revisi desain sebelum implementasi.
- Mencegah miskomunikasi antara tim desain dan developer.
- Hemat waktu dan biaya pengerjaan.
- Dapat diuji alur pengguna secara visual.

---

## Komponen: PromoCard

Berikut komponen sederhana bernama "PromoCard" yang ditulis dalam HTML dan CSS. Komponen ini berisi:
- Gambar promosi
- Judul promosi
- Tombol call-to-action "Daftar Sekarang"

### Cara Menjalankan:
1. Buka file `promo.html` di browser.
2. Komponen PromoCard akan muncul dengan gambar, teks, dan tombol.
3. Klik tombol untuk simulasi interaksi user.

---

### File dalam Repository:
- `promo.html`: File utama yang berisi struktur komponen PromoCard.
- `README.md`: Penjelasan studi kasus, manfaat prototipe, dan penjelasan kode.
