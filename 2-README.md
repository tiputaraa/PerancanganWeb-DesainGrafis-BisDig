
# Landing Page Promosi - PromoCard

## Studi Kasus
Sebuah perusahaan ingin membuat landing page promosi dengan Framework dan sebelumnya meminta prototipe dalam Figma.

## Penjelasan
### Manfaat Prototipe
- **Validasi Awal:** Memastikan alur dan struktur halaman sesuai ekspektasi klien.
- **Kolaborasi Tim:** Mempermudah komunikasi antar desainer dan developer.
- **Efisiensi Waktu:** Mengurangi revisi besar saat proses pengembangan.

### Komponen PromoCard
- Gambar promo
- Judul promo
- Deskripsi singkat
- Tombol "Daftar Sekarang"

## Contoh HTML PromoCard
File `promo.html` menampilkan komponen kartu promosi sederhana yang responsif dan menarik.
